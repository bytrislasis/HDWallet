Java Kurulum Rehberi
Adım Adım Java Kurulumu için Aşağıdaki Talimatları Takip Edin:

1. Java İndirme
Java'yı indirmek için Oracle Java SE 8 İndirme Sayfası adresine gidin.
İşletim sisteminize uygun Java sürümünü seçin (Windows, macOS, Linux).
İndirme işlemini başlatmak için "Accept License Agreement" seçeneğini işaretleyin ve indirme linkine tıklayın.


2. Java Kurulumu
İndirdiğiniz Java yükleme dosyasını açın.
Kurulum sihirbazındaki talimatları takip ederek Java'yı bilgisayarınıza kurun.


3. Java Kurulumunu Kontrol Etme
Kurulum tamamlandıktan sonra, Java'nın başarılı bir şekilde kurulup kurulmadığını kontrol etmek için bilgisayarınızda bir Komut İstemcisi veya Terminal penceresi açın.
Komut satırına java -version yazarak Enter tuşuna basın.
Eğer Java başarılı bir şekilde kurulmuşsa, yüklü Java sürümü hakkında bilgi alacaksınız.


4. SatoshiTURK.jar Dosyasını Çalıştırma
Java başarıyla kurulduktan sonra, SatoshiTURK.jar dosyasını çalıştırmak için, dosyanın bulunduğu klasöre gidin.
Dosyayı çift tıklayarak veya komut satırı/terminal üzerinden java -jar SatoshiTURK.jar komutu ile çalıştırın.


Sorun Giderme
Eğer SatoshiTURK.jar dosyasını çalıştıramıyorsanız, bilgisayarınızda doğru Java sürümünün kurulu olduğundan emin olun.
Kurulum veya çalıştırma sırasında herhangi bir sorunla karşılaşırsanız, SatoshiTURK Destek Forumu adresinden yardım isteyebilirsiniz.
